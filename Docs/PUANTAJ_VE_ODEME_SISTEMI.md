# Puantaj ve Aylık Ödeme Sistemleri

## ?? 1. Puantaj Sistemi (Personel Çalışma Takibi)

### Özellikler

#### **Aylık Puantaj:**
- Çalışılan gün sayısı
- Fazla mesai saatleri
- İzin ve mazeret günleri
- Brüt maaş hesaplama
- Yemek, yol, prim ödemeleri
- SGK kesintisi (%14)
- Gelir vergisi (%15)
- Damga vergisi (%0.759)
- Net ödeme hesaplama

#### **Günlük Puantaj:**
- Her gün için detaylı kayıt
- Çalıştı/Çalışmadı
- Fazla mesai saatleri
- İzin/Mazeret durumu
- Sefer/Güzergah bağlantısı

### Kullanım

```csharp
// Puantaj oluştur
var puantaj = new PersonelPuantaj
{
    PersonelId = 1,
    Yil = 2026,
    Ay = 3,
    CalisilanGun = 26,
    BrutMaas = 15000m
};
await _puantajService.CreateOrUpdatePuantajAsync(puantaj);

// Otomatik hesapla
await _puantajService.HesaplaAsync(puantaj.Id);

// Excel export
byte[] puantajExcel = await _puantajService.ExportPuantajListesiAsync(firmaId, 2026, 3);
byte[] vakifbankExcel = await _puantajService.ExportVakifbankOdemeListesiAsync(firmaId, 2026, 3);
```

## ?? 2. Aylık Ödeme Tablosu Sistemi

### Özellikler

#### **Ödeme Planı:**
- Ödeme adı (Kira, Kredi, vb.)
- Aylık tutar
- Ödeme günü (1-31)
- Başlangıç/Bitiş tarihi
- Otomatik kayıt oluşturma

#### **Gerçekleşen Ödemeler:**
- Planlanan tutar
- Ödenen tutar
- Ödeme tarihi
- Durum (Bekleniyor, Ödendi, Gecikti)

### Excel Çıktıları

#### **1. Aylık Ödeme Tablosu**
```
?????????????????????????????????????????????????????????????
?           AYLIK ÖDEME TABLOSU - 2026                     ?
?????????????????????????????????????????????????????????????
? Ödeme Türü      | Ocak | Şubat | Mart | ... | Toplam    ?
? Ofis Kirası     | 5000 | 5000  | 5000 | ... | 60000     ?
? Elektrik        | 800  | 900   | 850  | ... | 10200     ?
? Kredi Taksiti   | 3200 | 3200  | 3200 | ... | 38400     ?
?????????????????????????????????????????????????????????????
? TOPLAM          | 9000 | 9100  | 9050 | ... | 108600    ?
?????????????????????????????????????????????????????????????
```

**Tüm Firmalar:**
- Firma kısıtlaması YOK
- Tüm firmalar bir arada
- Firma bazında gruplama mevcut

#### **2. Aylık Ödeme Listesi**
```

? Şube  | Kira  | 5   | 3000      | 3000   | ?        ?
?????????????????????????????????????????????????????????
```

### Kullanım

```csharp
// Ödeme planı oluştur
var plan = new AylikOdemePlani
{
    FirmaId = 1,
    OdemeAdi = "Ofis Kirası",
    Turu = OdemeTuru.Kira,
    AylikTutar = 5000m,
    OdemeGunu = 5,
    OtomatikKayitOlustur = true
};
await _aylikOdemeService.CreatePlanAsync(plan);

// Ay için otomatik kayıtlar oluştur
await _aylikOdemeService.OtomatikOdemeKayitlariOlusturAsync(firmaId, 2026, 3);

// Excel export
byte[] aylikTablo = await _aylikOdemeService.ExportAylikOdemeTablosuAsync(2026, 3);
byte[] yillikTablo = await _aylikOdemeService.ExportYillikOdemeTablosuAsync(2026);
```
